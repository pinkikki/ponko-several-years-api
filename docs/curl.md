# curlメモ

| オプション | 内容                     |
|------------|--------------------------|
| I          | レスポンスヘッダ取得     |
| X          | メソッドの指定           |
| L          | リダイレクトさせる       |
| H          | ヘッダの指定             |
| A          | ユーザエージェントの指定 |
| F          | ファイルのアップロード   |
| d          | データの送信             |
